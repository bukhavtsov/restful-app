Вторая задача: написать restful приложение. Приложение должно уметь оперировать как минимум двумя сущностями в рамках CRUD операций.

Типы запросов:
GET /{сущность} - возвращает список всех сущностей
GET /{сущность}/{id} - возвращает одну запись (код 200), либо код 404, если не найдено
POST /{сущность} - создает сущность, если удачно - возвращает код 201
PUT /{сущность}/{id} - полностью обновляет запись
DELETE /{сущность}/{id} - удаляет запись

База - PostgreSQL. В репозитории должен быть скрипт для инициализации БД.

Для роутинга можно использовать gorilla/mux , для БД - либо стандартная библиотека + драйвер, либо upper.io, либо gorm.

Приложение должно логировать все входящие запросы (с пользовательской информацией) - написать middleware для этого.

Это первая часть. Продолжение после того как сделаете это.
